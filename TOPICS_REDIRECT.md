# 🎯 Guia Rápido - Sistema de Redirecionamento de Tópicos\n\n## O que foi adicionado\n\nVocê pode agora fazer os tópicos (seção \"Topics\") redirecionar para capítulos específicos dos livros.\n\n---\n\n## Como Usar\n\n### Passo 1: Abra a página da disciplina\n\nExemplo: `pages/biologia.html`\n\n### Passo 2: Encontre o bloco de configuração\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Nome do Tópico': { book: 1, chapter: 5 },\n    'Outro Tópico': { book: 2, chapter: 3, subsubject: 'Bio 1' }\n};\n```\n\n### Passo 3: Descomente e edite\n\nRemova os `/*` e `*/` para habilitar:\n\n```javascript\n// ANTES (comentado):\n/*\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5, subsubject: 'Bio 1' }\n};\n*/\n\n// DEPOIS (ativo):\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5, subsubject: 'Bio 1' }\n};\n```\n\n### Passo 4: Salve e teste\n\n- Salve a página\n- Recarregue no navegador\n- Clique em um tópico → deve ir para o livro/capítulo configurado\n\n---\n\n## Estrutura de Configuração\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Nome que aparece na página': {\n        book: 1,              // Número do livro (1-6, obrigatório)\n        chapter: 5,           // Número do capítulo (obrigatório)\n        subsubject: 'Bio 1'   // Nome da submatéria (opcional)\n    }\n};\n```\n\n### Campos obrigatórios:\n- **`book`** - Número do livro (1, 2, 3, 4, 5 ou 6)\n- **`chapter`** - Número do capítulo a destacar\n\n### Campo opcional:\n- **`subsubject`** - Nome da submatéria (se houver Bio 1, Bio 2, etc)\n\n---\n\n## Exemplos Reais\n\n### Exemplo 1: Biologia (com submatérias)\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5, subsubject: 'Bio 1' },\n    'Anatomia': { book: 1, chapter: 1, subsubject: 'Bio 1' },\n    'Genética': { book: 2, chapter: 8, subsubject: 'Bio 2' },\n    'Evolução': { book: 2, chapter: 15, subsubject: 'Bio 2' }\n};\n```\n\n**Resultado:**\n- Clicar em \"Fisiologia\" → vai para Livro 1, Capítulo 5 de Bio 1\n- Clicar em \"Genética\" → vai para Livro 2, Capítulo 8 de Bio 2\n\n### Exemplo 2: Matemática (sem submatérias)\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Álgebra': { book: 1, chapter: 1 },\n    'Geometria': { book: 1, chapter: 8 },\n    'Trigonometria': { book: 2, chapter: 3 },\n    'Funções': { book: 2, chapter: 15 },\n    'Cálculo': { book: 3, chapter: 5 }\n};\n```\n\n### Exemplo 3: História\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Brasil Colonial': { book: 1, chapter: 2 },\n    'Independência': { book: 1, chapter: 8 },\n    'República Velha': { book: 2, chapter: 5 },\n    'Era Vargas': { book: 2, chapter: 12 },\n    'Ditadura': { book: 3, chapter: 1 }\n};\n```\n\n---\n\n## O que Acontece ao Clicar\n\n1. **Visualmente**: O tópico fica com:\n   - Cursor de mão (pointer)\n   - Efeito de sombra ao passar o mouse\n   - Leve movimento para cima (hover effect)\n\n2. **Na navegação**: Redireciona para:\n   ```\n   livro.html?subject=biologia&book=1&subsubject=Bio%201&chapter=5\n   ```\n   \n3. **Na página de livro**: \n   - Abre o livro especificado\n   - Vai para o capítulo (futuro: scroll automático)\n   - Mostra todos os anexos daquele capítulo\n\n---\n\n## Dicas Importantes\n\n### ✅ Nomes devem corresponder exatamente\n\nO nome em `topicsRedirectConfig` deve ser **idêntico** ao nome do tópico na página:\n\n```html\n<!-- Em pages/biologia.html -->\n<div class=\"topic-item\"><h3>Fisiologia</h3><p>...</p></div>\n\n<!-- Em window.topicsRedirectConfig -->\n'Fisiologia': { book: 1, chapter: 5 }  // ✅ Correspondência exata\n'fisiologia': { book: 1, chapter: 5 }  // ❌ NÃO funciona (minúscula)\n'Fisiolog': { book: 1, chapter: 5 }    // ❌ NÃO funciona (faltam letras)\n```\n\n### ✅ Números de capítulo devem existir\n\nSe um livro tem 24 capítulos, configure capítulos 1-24:\n\n```javascript\n'Tópico': { book: 1, chapter: 100 }  // ❌ Não existe capítulo 100\n'Tópico': { book: 1, chapter: 15 }   // ✅ Capítulo válido\n```\n\n### ✅ Pode deixar alguns tópicos sem redirecionamento\n\n```javascript\n// Alguns tópicos são configurados, outros não\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5 },\n    'Genética': { book: 2, chapter: 8 }\n    // 'Anatomia' não está aqui → continua estático\n};\n```\n\n---\n\n## Casos de Uso\n\n### Caso 1: Página com tópicos estáticos\n\n**Situação:** Você não quer fazer os tópicos clicáveis.\n\n**Solução:** Deixe o bloco comentado (padrão)\n\n```javascript\n// Não descomente isto\n/*\nwindow.topicsRedirectConfig = { ... };\n*/\n```\n\n### Caso 2: Alguns tópicos clicáveis, outros não\n\n**Situação:** Você quer que apenas alguns tópicos redirecione.\n\n**Solução:** Configure apenas os que precisa\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5 }\n    // 'Genética' não está aqui\n};\n```\n\n### Caso 3: Tópicos em submatérias diferentes\n\n**Situação:** Um tópico pode estar em diferentes submatérias.\n\n**Solução:** Configure com `subsubject`\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Fisiologia': { book: 1, chapter: 5, subsubject: 'Bio 1' },\n    'Genética': { book: 2, chapter: 8, subsubject: 'Bio 2' }\n};\n```\n\n---\n\n## Testando a Configuração\n\n### Passo 1: Abra o console (F12)\n\n### Passo 2: Verifique se a configuração foi carregada\n\n```javascript\nconsole.log(window.topicsRedirectConfig);\n```\n\nDeve mostrar algo como:\n```javascript\n{\n    'Fisiologia': { book: 1, chapter: 5, subsubject: 'Bio 1' },\n    'Genética': { book: 2, chapter: 8, subsubject: 'Bio 2' }\n}\n```\n\n### Passo 3: Clique em um tópico\n\nDeve ir para a URL correta:\n```\nlivro.html?subject=biologia&book=1&subsubject=Bio%201&chapter=5\n```\n\n### Passo 4: Se não funcionar\n\n**Verificar:**\n- O nome no tópico é exatamente igual? (maiúsculas/minúsculas importam)\n- O `window.topicsRedirectConfig` não está comentado?\n- Recarregou a página depois de editar?\n- Abriu o console e viu erros?\n\n---\n\n## Adicionando a Mais Disciplinas\n\nO procedimento é o mesmo para qualquer disciplina.\n\n**Exemplo para Português:**\n\nEm `pages/portugues.html` (ou seu arquivo equivalente):\n\n```javascript\nwindow.topicsRedirectConfig = {\n    'Pronomes': { book: 1, chapter: 3 },\n    'Verbos': { book: 1, chapter: 8 },\n    'Orações': { book: 2, chapter: 5 },\n    'Concordância': { book: 2, chapter: 12 }\n};\n```\n\n---\n\n## Removendo o Sistema\n\nSe quiser desabilitar:\n\n1. Comente o bloco `window.topicsRedirectConfig`\n2. Ou remova-o completamente\n3. Recarregue a página\n\nOs tópicos voltarão a ser estáticos (sem interação).\n\n---\n\n## Resumo\n\n| Ação | Código |\n|------|--------|\n| **Habilitar redirecionamento** | Descomente `window.topicsRedirectConfig` |\n| **Adicionar um tópico** | `'Nome': { book: 1, chapter: 5 }` |\n| **Com submatéria** | `'Nome': { book: 1, chapter: 5, subsubject: 'Bio 1' }` |\n| **Deixar estático** | Não configure nele `topicsRedirectConfig` |\n| **Desabilitar tudo** | Comente o bloco inteiro |\n\n---\n\n## Suporte\n\nTodas as disciplinas em `pages/` têm este bloco comentado, pronto para usar:\n\n- ✅ matematica.html\n- ✅ biologia.html  \n- ✅ E todas as outras 13 disciplinas\n\nBasta descomente e edite conforme necessário!\n\n
