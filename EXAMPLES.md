\"Álgebra\" com 20 capítulos  \n- Livro 3: \"Geometria\" com 16 capítulos\n- Livro 4-6: Padrão\n\n### Solução\n\n**Arquivo:** `pages/matematica.html` (dentro do bloco `window.subjectBookConfig`)\n\n```javascript\nwindow.subjectBookConfig = {\n    subsubjects: {\n        'Mat 1': {\n            // Personalizar alguns livros\n            books: {\n                1: { title: 'Números e Conjuntos', chapters: 18 },\n                2: { title: 'Álgebra', chapters: 20 },\n                3: { title: 'Geometria', chapters: 16 },\n                4: { title: 'Livro 4', chapters: 24 },  // Padrão\n                5: { title: 'Livro 5', chapters: 24 },  // Padrão\n                6: { title: 'Livro 6', chapters: 24 }   // Padrão\n            },\n            attachments: { /* ... */ }\n        },\n        'Mat 2': {\n            // Sem books: recebe os 6 padrão automaticamente\n            attachments: { /* ... */ }\n        }\n    }\n};\n```\n\n### Resultado\n- Mat 1 tem livros com nomes e capítulos customizados\n- Mat 2 recebe os 6 livros padrão (Livro 1-6 com 24 capítulos cada)\n- URLs geradas corretamente para cada livro\n\n---\n\n## Exemplo `style.css`\n\n**Antes (azul):**\n```css\n:root {\n    --primary-dark: #1a3a52;   /* Azul escuro */\n    --primary-medium: #2c5282; /* Azul médio */\n    --primary-light: #3498db;  /* Azul claro */\n}\n```\n\n**Depois (verde):**\n```css\n:root {\n    --primary-dark: #1b5e20;   /* Verde escuro */\n    --primary-medium: #2d7a50; /* Verde médio */\n    --primary-light: #27ae60;  /* Verde claro */\n}\n```\n\n**Ou editar diretamente todos os usos:**\n\n```css\n/* Encontre e substitua em style.css: */\n#1a3a52 → #1b5e20  (azul escuro → verde escuro)\n#2c5282 → #2d7a50  (azul médio → verde médio)\n#3498db → #27ae60  (azul claro → verde claro)\n```\n\n**Ferramenta:** Use Ctrl+H (Find and Replace) em VS Code:\n- Find: `#1a3a52`\n- Replace All: `#1b5e20`\n\n### Resultado\n- Navbar: verde escuro\n- Botões e destaques: verde médio/claro\n- Barras laterais: verde claro\n- Todo o site muda de tema automaticamente\n\n---\n\n## Dúvidas Frequentes\n\n### P: Como adiciono um anexo só para um capítulo específico?\n\n**R:** Edite a configuração da disciplina:\n```javascript\nattachments: {\n    1: {        // Livro 1\n        5: [    // Apenas capítulo 5\n            {\n                label: 'Material',\n                viewUrl: 'url',\n                downloadUrl: 'url'\n            }\n        ]\n    }\n}\n```\nOutros capítulos do Livro 1 não terão anexos.\n\n---\n\n### P: Posso ter mais de 6 livros em uma disciplina?\n\n**R:** Sim! Adicione mais livros no bloco `books`:\n```javascript\nbooks: {\n    1: { title: 'Livro 1', chapters: 24 },\n    // ... até 6\n    7: { title: 'Livro 7 - Avançado', chapters: 20 },\n    8: { title: 'Livro 8 - Desafios', chapters: 15 }\n}\n```\nAparecerão automaticamente na navegação.\n\n---\n\n### P: Quais URLs são suportadas para anexos?\n\n**R:** Qualquer URL funciona:\n- **Local:** `../docs/arquivo.pdf`\n- **Externo:** `https://youtube.com/watch?v=...`\n- **Relativo:** `./materiais/apostila.pdf`\n- **Absoluto:** `/docs/materiais/apostila.pdf`\n\n---\n\n### P: Posso remover o localStorage para resetar tudo?\n\n**R:** Sim, no console do navegador:\n```javascript\nObject.keys(localStorage)\n    .filter(k => k.startsWith('subjectBookConfig:'))\n    .forEach(k => localStorage.removeItem(k));\n```\nDepois recarregue a página.\n\n---\n\n### P: Como faço backup das configurações?\n\n**R:** Exporte do localStorage:\n```javascript\nconst backup = {};\nObject.keys(localStorage)\n    .filter(k => k.startsWith('subjectBookConfig:'))\n    .forEach(k => backup[k] = localStorage[k]);\nconsole.log(JSON.stringify(backup));\n// Copie e salve em um arquivo .json\n```\n\nPara restaurar:\n```javascript\nconst backup = { /* cole aqui */ };\nObject.entries(backup).forEach(([k, v]) => localStorage[k] = v);\n```\n\n---\n\n### P: Preciso usar um
